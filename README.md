# Emotion Detection using OpenCV & Deep Learning

##  Overview
This project detects human emotions in real-time using a webcam.

##  Features
- Real-time face detection
- Emotion classification (7 classes)
- CNN-based deep learning model
- Data augmentation & class balancing

##  Tech Stack
- Python
- OpenCV
- TensorFlow / Keras
- NumPy, Pandas

##  Model Performance
- Accuracy: ~70–80%
- Dataset: FER2013

##  How to Run

```bash
install requirement librararies then
python realtime_detection.py
